# Oasis Infobyte Web Development Internship (OIBSIP)

This repository contains the projects completed as part of the **Web Development and Designing Internship** at [Oasis Infobyte](https://oasisinfobyte.com/).

## 📌 Projects Included

### 1. [Task 1: VPN Landing Page](./task1/)
A responsive landing page for a fictional VPN service (ExpressVPN clone).
- **Technologies Used:** HTML5, CSS3
- **Features:** Clean hero section, responsive feature grid layout, server statistics showcase, and modern user interface.

### 2. [Task 2: Personal Portfolio](./task2/)
A fully responsive, professional personal portfolio website designed to showcase skills, projects, and contact information. 
- **Technologies Used:** HTML5, CSS3 (with Tailwind CSS setup), JavaScript
- **Features:** Interactive navigation menu, smooth scrolling, project gallery layout, testimonials carousel, and a contact form UI.

### 3. [Task 3: Temperature Converter](./task3/)
A functional web application to convert temperatures between Celsius and Fahrenheit.
- **Technologies Used:** HTML5, CSS3, JavaScript
- **Features:** Instant conversion logic, dropdown selection for input type, neat and minimalistic UI.

## 🚀 How to Run

1. Navigate to any of the task directories (e.g., `task1`, `task2`, or `task3`).
2. Open the `index.html` file in your preferred web browser to view that specific project locally.
